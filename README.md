# wmlang.js

## Demo

http://tools.bitfertig.de/wmlang.js/

## Install

```bash
npm i @dipser/wmlang.js
```

## Usage

```html
<script type="module">
import { month, weekday } from '@dipser/wmlang.js';
console.log( weekday(1, 'de') ); // => Montag
console.log( month(1, 'de') ); // => Januar
</script>
```
